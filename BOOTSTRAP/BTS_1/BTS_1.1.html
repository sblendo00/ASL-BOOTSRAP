A cosa serve bootstrap?

Utilizzabile dagli sviluppatori di tutto il mondo come base per la realizzazione di interfacce web. Al momento è considerato il più efficiente ed il più utilizzato framework per rendere i siti web responsive grazie alla sua facilità d'uso.

Cosa devi importare per creare una pagina web con bootstrap

CSS (Bootstrap) , jQuery e javascript

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

È una "rete per la consegna di contenuti" che permette di importare librerie direttamente dalla pagina HTML

A cosa serve questo attributo del tag meta?content="width=device-width, initial-scale=1

Serve per fare adattare lo zoom e la pagina alla larghezza del dispositivo

Che differenza c’è tra Container e Container Fluid

Il container ha una larghezza fissa, la larghezza del container-fluid invece si adatta alla pagina
